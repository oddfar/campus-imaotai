## 项目介绍

i茅台app，每日自动预约茅台，可添加多个用户，可选本市出货量最大的门店，或预约你的位置附近门店

软件会在每日9点05开始批量预约，并 [pushplus](https://www.pushplus.plus/) 推送消息

此项目使用 **Campus** 进行编写：<https://github.com/oddfar/campus>
